IoT Smart Home di Cisco Packet Tracer

Project Smart Home Sederhana menggunakan Internet of Things

Project Overview

Pada proyek ini, dibuat sebuah simulasi Smart Home menggunakan aplikasi Cisco Packet Tracer dengan memanfaatkan fitur IoT (Internet of Things). Tujuannya adalah untuk memahami bagaimana perangkat rumah tangga dapat saling terhubung dan dikontrol melalui jaringan.

Topologi awal tanpa koneksi ke Home Gateway

1. Perangkat yang Digunakan

Dalam simulasi ini digunakan beberapa perangkat IoT, yaitu:

Light (Lampu)

Dapat dinyalakan, dimatikan, atau diatur tingkat kecerahannya.

Ceiling Fan (Kipas Angin)

Dapat dikontrol dengan mode Low atau High.

Door (Pintu)

Berfungsi sebagai akses masuk rumah yang dapat dipantau.

Motion Detector

Mendeteksi pergerakan untuk otomatisasi perangkat.

Smartphone

Digunakan untuk mengontrol seluruh perangkat melalui aplikasi IoT Monitor.

Home Gateway (DLC100)

Penghubung utama antara perangkat IoT dengan jaringan nirkabel.

2. Konfigurasi Jaringan

1

Rename HomeGateway

Click alat Home gateway lalu pergi ke config pada bagian Wireless setting ssid bebas contoh RafaHome.

Setting SSID pada Home Gateway
2

Konfigurasi IoT Server

Pada menu konfigurasi masing-masing perangkat (lampu,fan,door, etc.), click alatnya lalu pergi ke pengaturan config lalu pada opsi IoT Server → Home Gateway diaktifkan agar perangkat terhubung ke home gateway.

Alat - config - global setting - scroll - pada box IoT server
3

Koneksi WiFi Smartphone

Smartphone juga dikonfigurasi agar terhubung ke jaringan WiFi (SSID: Rafahome).

Phone - config - Wireless
4

Buka IoT Monitor

Setelah terhubung, buka aplikasi IoT Monitor pada smartphone.

Aplikasi IoT Monitor pada smartphone
5

Login ke IoT Monitor

Smartphone Jika sudah di set maka ip akan terisi otomatis oleh dhcp maka tinggal click Login.

Login ke aplikasi IoT Monitor

3. Kontrol dan Monitoring

Kontrol Real-time

Dari IoT Monitor, pengguna dapat mengatur status perangkat (On/Off, Dim/High).

Monitoring Sensor

Sensor gerak dan pintu bisa dipantau melalui aplikasi yang sama.

Otomatisasi

Contoh: Lampu bisa dinyalakan melalui smartphone, atau kipas angin bisa diatur kecepatannya.

4. Hasil Simulasi

Dengan konfigurasi ini, pengguna dapat:

Mengontrol lampu dan kipas angin langsung dari smartphone
Memantau kondisi pintu dan sensor gerak secara real-time
Membuat kondisi otomatisasi, misalnya ketika sensor gerak mendeteksi pergerakan, lampu menyala otomatis
✅ Hasil akhir: Test berhasil membuat fan berputar
